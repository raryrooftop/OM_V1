# 재활치료센터 다면 역량평가 시스템 — 내부 개발용 구현 설계서 (기초 구성)

> **구현 전제:** GCP 풀스택 환경에서 **Antigravity**로 개발
> **버전:** 1.0 · 작성: 재활치료센터
> **성격:** 세부 기술 구현이 아니라 *무엇을 만들 것인가*(평가 내용·평가 방법·점수 계산·화면 구성)를 정의하는 기초 설계서

---

## 1. 문서의 목적과 범위

본 문서는 Antigravity로 평가 웹앱을 구현하기 위한 **기초 구성 설계서**다. 세부 기술 구현(코드·인프라 설정)이 아니라, *무엇을 만들 것인가*를 평가 내용·평가 방법·점수 계산·화면 구성 중심으로 정의한다. 디자인 시스템과 세부 기술 스택은 후속 단계에서 확정한다.

| 구분 | 내용 |
|---|---|
| **포함 범위** | 사용자·권한, 평가 차수, 평가 내용(차원), 다면 평가자 매핑, 점수 계산 로직, 화면 4종 구성, 데이터 항목(기초) |
| **제외 범위** | 디자인 시스템(UI 스타일), 화면 픽셀 설계, DB 물리 스키마, 인증·배포 상세, API 명세 — 모두 후속 단계 |
| **구현 전제** | GCP 풀스택(App Engine/Firestore 등) 환경에서 Antigravity로 개발 |

## 2. 사용자와 권한

### 2-1. 직급·직책

구성원은 **직급(팀원 등급)**과 **직책(관리 보직)**으로 구분된다. 한 사람은 직급을 반드시 갖고, 직책은 선택적으로 갖는다(예: 책임치료사이면서 팀매니저).

| 구분 | 명칭 | 역할 |
|---|---|---|
| **직급** | 치료사 | 1~2년차 팀원 |
| | 선임치료사 | 3~4년차 팀원 |
| | 책임치료사 | 5년차 이상 팀원 |
| **직책** | 팀매니저 | 팀 관리 (기존 부팀장/팀장) |
| | 파트매니저 | 파트 관리 (기존 실장/파트장) |
| | 시니어매니저 | 센터 총괄·인사 (기존 차장/센터장) |

### 2-2. 시스템 권한(Role)

화면 접근 권한을 세 단계로 둔다. 권한은 직책에 따라 부여된다.

| 권한 | 접근 가능 화면 |
|---|---|
| **팀원** | ① 평가 수행 화면(배정된 평가) · ② 본인 결과 확인 화면 |
| **매니저**<br>(팀매니저·파트매니저) | 팀원 권한 + 자신에게 배정된 추가 평가(팀원 평가·동료평가·상향평가) |
| **시니어매니저** | 매니저 권한 + ③ 대시보드(전 구성원 결과 열람) + ④ 차수·배정 관리(운영 전권) |

> ※ 시니어매니저도 피평가자이므로, 시니어매니저에 대한 평가는 외부(상위 경영진) 또는 자기평가+파트매니저 상향평가로 별도 처리한다(운영 규정에서 확정).

## 3. 평가 차수(年 단위) 운영

평가는 매년 일정 기간 ‘차수’를 열어 진행한다. 차수는 다음 상태를 거친다.

| 상태 | 의미 | 주요 동작 |
|---|---|---|
| **준비 (Draft)** | 차수 개설·배정 중 | 시니어매니저가 대상자·평가자 배정 |
| **진행 (Open)** | 평가 입력 기간 | 평가자들이 평가 수행, 자기평가 제출 |
| **마감 (Closed)** | 입력 종료 | 점수 계산·결과 산출, 회수율 검증 |
| **공개 (Released)** | 결과 열람 단계 | 면담 연계 후 본인·대시보드 공개 |

지난 차수의 결과는 영구 보존되어, 대시보드에서 연도별 추이로 비교된다.

## 4. 평가 내용 — 역량 차원과 척도

평가 항목은 기존 「역량평가서(BARS)」의 **16개 차원을 그대로 탑재**한다. 각 차원은 5점(탁월)~1점(미흡)이며, 점수마다 회복기 재활 현장 행동 예시가 붙는다.

| 역량군 | 차원 | 대상 |
|---|---|---|
| **공통역량** | 4개 | 전 구성원 |
| **직무역량** | 7개 | 전 구성원(직종 특화는 메모로 보완) |
| **리더십역량** | 5개 | 직책자(매니저)만 |

**역량군별 차원 목록**

- **공통역량(4):** 성실·직업윤리, 환자중심, 의사소통, 협업
- **직무역량(7):** 환자 평가·기능검사, 치료계획, 치료 수행, 환자교육, 환자안전, 기록·청구, 자기개발
- **리더십역량(5):** 코칭·육성, 성과·업무관리, 갈등관리, 의사결정, 자원·운영관리

> 세부 16개 차원의 5점 행동기준 전문은 **본 문서 [부록 A](#부록-a-평가-차원-전문--bars-16개-차원-5점-행동기준)**에 수록하며, 본 시스템은 이를 마스터 데이터로 탑재한다.

### 4-1. 근거·메모 입력 규칙

점수의 신뢰성을 위해 극단 점수에는 근거 입력을 강제한다.

| 점수 | 메모 | 입력 안내(placeholder) |
|---|---|---|
| **1·2점** | 필수 | 부족한 점과 개선 방향을 적어 주세요 |
| **3점** | 선택 | (선택) 특이사항이 있으면 적어 주세요 |
| **4·5점** | 필수 | 그렇게 판단한 구체적 행동 사례를 적어 주세요 |

> ※ 필수 메모는 **최소 10자 이상**이어야 제출되며, 미충족 시 해당 문항이 표시되고 제출이 차단된다.

## 5. 평가 방법 — 다면 평가자 매핑

평가 주체는 본인·동료·팀원(상향)·상급자다. 피평가자 유형에 따라 평가자가 달라진다.

| 피평가자 | 본인 | 동료(횡적) | 팀원(상향) | 상급자 |
|---|---|---|---|---|
| **일반 치료사** | 격차용 | 같은 팀 동료 | — | 직속 팀매니저 |
| **팀매니저** | 격차용 | 같은 파트 타 팀매니저 | 본인 팀원 | 파트매니저 |
| **파트매니저** | 격차용 | 타 파트매니저 | 파트 구성원 | 시니어매니저 |
| **언어재활 치료사** | 격차용 | 교차동료 10명 + 타파트 팀매니저 | — | 언어 매니저/시니어매니저 |

> ※ 스포츠재활(1팀 7명)의 상급자는 파트매니저다.
> ※ 언어재활 교차동료는 성인물리·작업치료 치료사 중 **각 5명을 무작위·익명 추출**한다.

## 6. 점수 계산 로직

점수는 **‘평가자 → 역량군 → 종합’의 2단계 가중**으로 계산한다. 자기평가는 계산에서 제외하고 격차 표시에만 쓴다.

### 6-1. 1단계 — 역량군 점수(평가자 가중)

```
역량군 점수 = Σ ( 평가자 유형 평균 × 평가자 비중 )
```

평가자 비중(역량군별 차등)은 다음과 같다.

**① 일반 치료사(비직책)**

| 역량군 | 동료 | 상급자 |
|---|---|---|
| 공통역량 | 50% | 50% |
| 직무역량 | 25% | 75% |

**② 직책자(팀매니저·파트매니저)**

| 역량군 | 동료 | 팀원(상향) | 상급자 |
|---|---|---|---|
| 공통역량 | 30% | 20% | 50% |
| 직무역량 | 20% | — | 80% |
| 리더십역량 | — | 55% | 45% |

**③ 언어재활 치료사(교차평가 특례)**

| 역량군 | 교차동료(10) | 타파트 팀매니저 | 상급자 |
|---|---|---|---|
| 공통역량 | 50% | 20% | 30% |
| 직무역량(ST) | — | — | **100%** |

> 타 직종은 언어치료의 전문 술기를 판정할 수 없으므로, 교차평가는 공통역량(협업·태도)에만 적용하고 직무역량은 상급자가 전담한다.

### 6-2. 2단계 — 종합 점수(트랙별 역량군 가중)

```
종합 = 공통점수 × 공통가중 + 직무점수 × 직무가중 + 리더십점수 × 리더십가중
```

트랙은 직급·직책으로 자동 결정되며, 역량군 가중은 다음과 같다.

| 트랙 | 공통 | 직무 | 리더십 |
|---|---|---|---|
| 실무트랙 (치료사·선임치료사) | 30% | 70% | — |
| 전문가트랙 (책임치료사) | 25% | 75% | — |
| 관리트랙 (직책 보유자) | 15% | 25% | 60% |

> **계산 예)** 팀매니저 — 공통 3.71 · 직무 3.90 · 리더십 3.69
> → `3.71×0.15 + 3.90×0.25 + 3.69×0.60 ≈ 3.75` → **A등급**

### 6-3. 등급 환산

| 종합점수 | 등급 |
|---|---|
| 4.5 이상 | **S (탁월)** |
| 3.5 ~ 4.49 | **A (우수)** |
| 2.5 ~ 3.49 | **B (기대충족)** |
| 1.5 ~ 2.49 | **C (개선필요)** |
| 1.0 ~ 1.49 | **D (미흡)** |

### 6-4. 집계 보정 규칙

- **절사평균** — 동료·교차동료가 5명 이상이면 최고·최저 각 1개를 제외한 평균을 사용한다(친분·갈등 극단값 제거). 5명 미만이면 단순평균.
- **회수율** — 지정 평가자의 **60% 이상**이 제출해야 그 유형 점수가 유효하다. 미달 시 해당 유형 비중을 상급자 평가로 흡수한다.
- **소표본 마스킹** — 평가자가 **3명 미만**인 유형은 점수를 산출·표시하지 않고 ‘표본 부족’으로 처리한다(익명성 보호).
- **자기평가 격차** — 각 차원에서 `(상급자 점수 − 자기 점수)`를 격차로 계산해, 본인 결과 화면과 면담 자료로만 표시한다.

## 7. 화면별 구성(기초)

핵심 화면은 네 개다. 각 화면의 목적·구성 요소·동작을 정의한다. 디자인(스타일)은 후속 단계에서 입힌다.

### 7-1. ① 평가 수행 화면

- **사용 권한:** 팀원·매니저(평가자 전원)
- **목적:** 이번 차수에 내가 할 평가를 한곳에 모아 누락 없이 수행

**구성 요소**

- 상단: 현재 차수명·마감일·내 전체 진행률(예: 완료 7 / 전체 12)
- ‘할 일 목록’을 평가 유형별 섹션으로 구분: ① 나의 자기평가 ② 팀원 평가(n) ③ 동료평가(n) ④ 상향평가(n)
- 각 항목 카드: 피평가자 표시(익명 평가는 대상자명만), 상태 배지(미시작·작성중·완료), `[평가하기]` 버튼
- 평가 입력 패널: 역량군 → 차원 순으로 16개(직책자) 또는 11개(팀원) 차원, 각 차원에 5점 행동기준 표시 + 점수 선택 + 근거·메모
- 1·2·4·5점 선택 시 메모 필수(10자 이상) 검증, 미충족 항목 강조
- 임시저장 / 제출 버튼, 제출 후 수정 가능 여부는 차수 마감 전까지로 제한
- 익명 안내 문구: 동료·상향평가는 누가 평가했는지 결과에 연결되지 않음을 명시

**동작 흐름**

1. 로그인 → 내 차수의 ‘할 일 목록’ 표시
2. 항목 선택 → 평가 입력 패널에서 차원별 점수·메모 입력
3. 검증 통과 시 제출 → 목록의 상태가 ‘완료’로 변경
4. 모든 항목 완료 시 진행률 100%

### 7-2. ② 자기평가·결과 확인 화면

- **사용 권한:** 피평가자 본인
- **목적:** 본인 자기평가 확인 및 마감 후 유형별 결과·격차 확인

**구성 요소**

- 진행 중: 본인이 제출한 자기평가 점수·메모를 차원별로 다시 확인(수정은 마감 전까지)
- 공개(Released) 후: 종합점수·등급, 역량군별 점수 표시
- 유형별 결과: 동료 평균·상향 평균·상급자 평균을 분리 표시(개별 평가자 식별 불가)
- 자기-타인 격차: 차원별 `(타인평균 − 자기점수)`를 시각화해 성찰 포인트 강조(격차 2점 이상 표시)
- 소표본 마스킹: 평가자 3명 미만 유형은 ‘표본 부족’으로 점수 숨김
- 개발계획(IDP) 입력란: 강점·개선영역·차기 목표를 본인이 작성/면담 후 보완
- 공개 시점은 시니어매니저가 면담 연계로 제어(임의 조기 열람 차단)

**동작 흐름**

1. 진행 중 자기평가 확인/수정
2. 차수 공개 후 결과 화면 활성화
3. 유형별 점수·격차 확인 → 면담 → IDP 작성

### 7-3. ③ 시니어매니저 대시보드

- **사용 권한:** 시니어매니저 전용
- **목적:** 전 구성원의 결과를 인사 관점에서 확인·분석

**구성 요소**

- 차수 선택 + 파트·직급·직책 필터
- 구성원 목록: 종합점수·등급·회수율·완료여부를 표로 표시, 정렬·검색
- 개인 상세: 선택한 구성원의 역량군별 점수 + 유형별 분리(자기·동료·상향·상급자) — 개별 평가자는 식별 불가
- 차원별 보기: 16개 차원 점수와 근거 메모(익명) 열람
- 분포·비교: 파트별·직급별 평균 분포, 등급 분포(예: S/A/B/C/D 인원)
- 연도별 추이: 동일 구성원의 차수 간 종합점수 변화 그래프
- 내보내기: 인사 활용을 위한 결과 표 다운로드(권한 로그 기록)

**동작 흐름**

1. 차수 선택 → 구성원 목록 확인
2. 구성원 선택 → 유형별·차원별 상세 확인
3. 필요 시 분포·추이 분석 → 내보내기

### 7-4. ④ 차수·배정 관리 화면

- **사용 권한:** 시니어매니저(관리자)
- **목적:** 평가 차수 개설과 대상자·평가자 배정, 마감·산출 운영

**구성 요소**

- 차수 관리: 새 차수 개설(연도·기간), 상태 전환(준비→진행→마감→공개)
- 대상자 등록: 평가 대상 구성원 목록 관리(직급·직책·파트 정보)
- 평가자 배정: 규칙 기반 자동 배정(매핑 규칙) + 수동 보정. 언어재활 교차동료 무작위 추출 기능
- 진행 모니터링: 평가자별 회수율·미제출자 현황, 독려 알림(예: Telegram 연계 — 기존 자산 활용)
- 마감·산출: 마감 시 점수 자동 계산(2단계 가중·절사·회수율·마스킹 적용)
- 공개 제어: 면담 완료 후 결과 공개 전환

**동작 흐름**

1. 차수 개설 → 대상자·평가자 배정
2. 진행 기간 모니터링 → 마감
3. 산출 결과 확인 → 면담 → 공개 전환

## 9. 향후 결정·확정 필요 사항

- **디자인 시스템** — UI 스타일·컴포넌트(색·타이포·레이아웃). 구현 직전 선정.

- **시니어매니저 평가 방식** — 시니어매니저 본인에 대한 평가 주체·비중(상위 경영진 또는 파트매니저 상향).

## 부록 A. 평가 차원 전문 — BARS 16개 차원 (5점 행동기준)

본 부록은 시스템에 탑재하는 평가 문항의 전문이다. 각 차원은 5점(탁월)~1점(미흡)이며, 회복기 재활의료기관 현장 행동을 기준으로 작성되었다. **1·2·4·5점 선택 시 근거·메모 입력이 필수**(최소 10자)이며, 3점은 선택이다.

### 적용 대상 요약

| 역량군 | 차원 수 | 평가 대상 |
|---|:---:|---|
| 공통역량 | 4 | 전 구성원 |
| 직무역량 | 7 | 전 구성원 (직종 특화는 메모 보완) |
| 리더십역량 | 5 | 직책자(매니저)만 |

## A-1. 공통역량 (전 구성원 · 4개 차원)


#### 1. 성실·직업윤리

> **정의:** 치료 성실 수행, 환자 정보보호, 근태·직업윤리 준수

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 맡은 환자의 치료 시간·횟수를 빠짐없이 채우고, 환자 정보와 진료내용을 철저히 지키며, 비협조적이거나 까다로운 환자도 끝까지 책임지고 동료에게 본이 된다 |
| 4 (우수) | 치료 시간·횟수를 성실히 지키고, 환자 정보보호와 동의 절차를 한결같이 지킨다 |
| 3 (기대충족) | 정해진 치료 일정과 기본 직업윤리(정보보호·환자 존중)를 지킨다 |
| 2 (개선필요) | 치료 누락·지각이나 정보 관리에서 가끔 주의가 필요하다 |
| **1 (미흡)** | 무단 치료 누락이나 환자 정보 부주의 등 기본이 반복적으로 지켜지지 않는다 |

#### 2. 환자중심 치료

> **정의:** 환자의 회복 목표·생활여건을 고려해 치료에 반영

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 환자의 회복 목표와 퇴원 후 생활(재택복귀·시설 등)을 환자·보호자와 함께 정하고, 체력·인지·집안 환경까지 고려해 치료를 맞춰 간다 |
| 4 (우수) | 환자·보호자의 바람과 생활여건을 파악해 치료에 반영한다 |
| 3 (기대충족) | 환자의 요구를 듣고 무리 없는 선에서 치료에 반영한다 |
| 2 (개선필요) | 정해진 프로그램대로만 진행해 환자 사정 반영이 부족하다 |
| **1 (미흡)** | 환자가 무엇을 원하는지, 어떤 형편인지 거의 고려하지 않는다 |

#### 3. 의사소통

> **정의:** 환자·보호자·동료·타 직종과의 원활한 소통

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 환자·보호자에게 회복 정도와 주의사항을 알아듣기 쉽게 설명하고, 의사·간호·타 치료실과도 환자 상태를 막힘없이 주고받는다 |
| 4 (우수) | 환자·보호자, 동료와 필요한 내용을 분명하고 정중하게 주고받는다 |
| 3 (기대충족) | 업무에 필요한 의사소통을 무리 없이 한다 |
| 2 (개선필요) | 설명이 불분명하거나 인계가 부실해 오해·재확인이 잦다 |
| **1 (미흡)** | 의사소통 문제로 환자 혼선이나 업무 차질이 반복된다 |

#### 4. 협업·팀워크

> **정의:** 다학제 팀·치료실 운영에 협력

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 다학제 팀회의에서 환자 정보를 적극 공유하고, 바쁜 동료의 환자를 먼저 챙기며 치료실 분위기를 끌어올린다 |
| 4 (우수) | 팀회의·인계에 성실히 참여하고 동료와 잘 협력한다 |
| 3 (기대충족) | 맡은 일을 하고 요청하면 동료를 돕는다 |
| 2 (개선필요) | 협조가 수동적이거나 인계·정보 공유가 늦을 때가 있다 |
| **1 (미흡)** | 비협조적이어서 팀·치료실 운영에 부담을 준다 |

## A-2. 직무역량 (전 구성원 · 7개 차원)

> ※ 직종 특화 항목(PT: 도수·이동 / OT: ADL·인지 / ST: 의사소통·연하)은 해당 차원의 근거·메모에 직종 맥락으로 기재한다.


#### 1. 환자 평가·기능검사

> **정의:** FIM·K-MBI 등 기능평가와 MMT·ROM 등 검사 시행·해석

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | FIM·K-MBI 등 기능평가와 MMT·ROM 같은 검사를 정확히 시행하고, 상태가 복잡한 환자도 결과를 종합해 핵심 문제를 정확히 짚어낸다 |
| 4 (우수) | 기능평가와 도수검사를 정확히 시행하고 결과를 적절히 해석한다 |
| 3 (기대충족) | 정해진 평가도구를 절차대로 시행하고 기본 해석을 한다 |
| 2 (개선필요) | 평가 시행·해석에 오류가 있어 선임의 확인이 필요하다 |
| **1 (미흡)** | 평가가 부정확해 재검사·지도가 자주 필요하다 |

#### 2. 치료계획·목표설정

> **정의:** 평가에 근거한 단기·장기 목표와 치료계획 수립·수정

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 평가 결과와 환자 회복 목표를 바탕으로 현실적인 단기·장기 목표와 치료계획을 세우고, 회복 속도에 맞춰 계획을 능동적으로 고치며 그 이유를 설명한다 |
| 4 (우수) | 평가에 근거해 목표·치료계획을 세우고 경과에 따라 조정한다 |
| 3 (기대충족) | 표준 프로토콜에 따라 치료계획을 세운다 |
| 2 (개선필요) | 계획이 형식적이고 경과 반영·수정이 늦다 |
| **1 (미흡)** | 평가와 치료계획이 따로 놀아 목표가 불분명하다 |

#### 3. 치료 수행·술기

> **정의:** 안전·정확한 치료 수행과 환자 반응 대응

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 치료를 안전하고 정확하게 진행하면서 환자 반응(통증·피로·혈압 등)에 맞춰 그때그때 강도를 조절하고, 까다로운 케이스도 능숙히 다룬다 |
| 4 (우수) | 치료를 안전·정확하게 수행하고 환자 반응에 잘 대응한다 |
| 3 (기대충족) | 표준 치료기법을 안전하게 수행한다 |
| 2 (개선필요) | 치료 수행이 미숙해 지도·보완이 필요하다 |
| **1 (미흡)** | 치료 중 안전·정확성 문제가 반복된다 |

#### 4. 환자·보호자 교육

> **정의:** 자가운동·가정 내 케어 교육(재택복귀 대비)

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 환자·보호자가 알아듣기 쉽게 자가운동·주의사항·집에서의 케어 방법을 가르치고, 직접 해 보게 하며 퇴원 후에도 이어가도록 이끈다 |
| 4 (우수) | 자가운동·주의사항을 환자·보호자가 이해하도록 분명히 교육한다 |
| 3 (기대충족) | 필요한 교육·설명을 정확히 제공한다 |
| 2 (개선필요) | 설명이 부족하거나 어려워 환자가 따라 하기 힘들다 |
| **1 (미흡)** | 교육이 부실해 자가관리·보호자 케어가 잘 되지 않는다 |

#### 5. 환자안전

> **정의:** 낙상·흡인·욕창 등 위험 확인과 이상징후 대응

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 치료 전 환자 상태·금기·낙상위험을 스스로 확인하고, 이상징후(혈압·통증·어지럼·흡인 등)를 빨리 알아채 바로 중단·보고하며, 재발을 막을 방법까지 제안한다 |
| 4 (우수) | 금기·위험요인을 확인하고 이상징후가 보이면 즉시 중단·보고한다 |
| 3 (기대충족) | 낙상·감염 등 기본 안전수칙을 지키고 분명한 이상징후는 보고한다 |
| 2 (개선필요) | 안전수칙은 대체로 지키나 사전 위험 확인을 자주 빠뜨린다 |
| **1 (미흡)** | 안전수칙을 자주 놓치거나 이상징후를 챙기지 못한다 |

#### 6. 치료기록·청구 연계

> **정의:** 전자차트 당일 기록과 상병·치료내역 정확 기재

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 치료 기록을 그날그날 정확히 남기고 상병·치료내역을 규정대로 기재해 청구·적정성평가에 문제가 없게 하며, 동료 기록까지 점검·보완해 준다 |
| 4 (우수) | 치료 기록을 제때 정확히, 규정에 맞게 작성한다 |
| 3 (기대충족) | 규정에 따라 필요한 기록을 작성한다 |
| 2 (개선필요) | 기록이 밀리거나 누락·오기가 종종 있다 |
| **1 (미흡)** | 기록 미비·지연이 반복돼 청구·평가에 지장을 준다 |

#### 7. 자기개발·전문성

> **정의:** 보수교육·새 기법 습득과 사례 공유

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 새로운 치료기법과 최신 지견을 익혀 치료에 적용하고, 사례를 정리해 동료와 나누며 치료실 수준을 함께 끌어올린다 |
| 4 (우수) | 보수교육·자료를 치료에 반영하며 꾸준히 공부한다 |
| 3 (기대충족) | 필수 보수교육을 이수하고 기본 실력을 유지한다 |
| 2 (개선필요) | 공부·자기개발이 수동적이고 새 지견 반영이 더디다 |
| **1 (미흡)** | 자기개발 노력이 거의 없다 |

## A-3. 리더십역량 (직책자 전용 · 5개 차원)

> ※ 팀매니저·파트매니저 등 직책 보유자만 평가한다. 팀원(비직책)은 평가하지 않는다.


#### 1. 팀원 코칭·육성

> **정의:** 치료사 지도와 성장 지원

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 치료사 한 명 한 명의 강점·부족한 점을 파악해 성장 방향을 잡아 주고, 평소 치료 장면을 보며 구체적으로 피드백·지도하며 교육 기회를 먼저 연결해 준다 |
| 4 (우수) | 정기적으로 피드백하고 팀원의 교육·성장 기회를 적극 챙긴다 |
| 3 (기대충족) | 요청하면 피드백을 주고 필요한 교육을 안내한다 |
| 2 (개선필요) | 지시는 하지만 일을 맡기길 꺼리고 후배 지도가 드물다 |
| **1 (미흡)** | 팀원 지도·피드백을 거의 하지 않는다 |

#### 2. 성과·업무관리

> **정의:** 환자 배정·치료 일정과 실적 관리

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 팀 목표와 환자 배정·치료 일정을 명확히 잡고, 치료 실적과 진행 상황을 수시로 챙기며 처지는 부분을 빨리 잡아 개선한다 |
| 4 (우수) | 팀 목표를 세우고 실적·진행을 정기적으로 점검·피드백한다 |
| 3 (기대충족) | 정해진 목표에 따라 팀 업무를 관리한다 |
| 2 (개선필요) | 목표·기준이 모호하거나 점검이 들쭉날쭉하다 |
| **1 (미흡)** | 업무관리가 되지 않아 팀 운영이 방치된다 |

#### 3. 갈등관리·조직운영

> **정의:** 갈등의 공정한 조정과 치료실 운영

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 팀원 간 갈등이나 타 치료실과의 마찰을 원인까지 파고들어 공정하게 풀고, 다시 생기지 않도록 규칙·분위기를 잡는다 |
| 4 (우수) | 팀 안의 갈등을 공정하게 다루고 원만히 조정한다 |
| 3 (기대충족) | 갈등이 생기면 개입해 기본적으로 정리한다 |
| 2 (개선필요) | 개입이 늦거나 한쪽에 치우쳐 뒷말이 남는다 |
| **1 (미흡)** | 갈등을 방치하거나 더 키운다 |

#### 4. 의사결정·문제해결

> **정의:** 현장 상황에 맞는 시의적절한 판단

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 현장 상황과 데이터를 종합해 제때 판단하고, 애매한 상황에서도 책임지고 결정하며 결과를 되짚어 보완한다 |
| 4 (우수) | 필요한 정보를 모아 합리적으로, 제때 결정한다 |
| 3 (기대충족) | 정해진 범위 안에서 필요한 결정을 한다 |
| 2 (개선필요) | 결정을 미루거나 근거가 부족한 판단을 할 때가 있다 |
| **1 (미흡)** | 결정을 피하거나 잘못된 판단으로 업무에 지장을 준다 |

#### 5. 자원·운영관리

> **정의:** 인력·치료실 일정·장비의 효율적 운영

| 점수 | 행동기준 |
|:---:|---|
| **5 (탁월)** | 치료사 인력과 치료실 일정·장비를 효율적으로 운영해 환자 대기와 치료 흐름을 원활히 하고, 개선거리를 먼저 찾아 손본다 |
| 4 (우수) | 인력·일정을 효율적으로 운영하고 자원을 적절히 배분한다 |
| 3 (기대충족) | 주어진 자원으로 정상 운영을 유지한다 |
| 2 (개선필요) | 자원 배분이 비효율적이거나 운영 차질이 잦다 |
| **1 (미흡)** | 운영관리 미흡으로 치료실에 지장이 반복된다 |
