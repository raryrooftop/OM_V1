# OM_V1 — 재활치료센터 조직개편 및 다면 역량평가 시스템

재활치료센터의 **직급·직책 분리 조직계층 개편**과 **다면 역량평가 시스템(BARS 기반)** 도입을 위한 산출물 모음입니다.

## 바로 보기

- **[제안서 (index.html)](./index.html)** — 경영진 결재용 통합 제안서. 조직계층·다면평가 상세·16개 차원 행동기준·점수 산정 다이어그램 포함. 브라우저에서 열면 Pretendard 글꼴과 다이어그램이 적용됩니다.

## 폴더 구성

```
OM_V1/
├── index.html              제안서(최신, 다이어그램 3종 포함)
├── docs/                   기획·설계 문서
│   ├── 01_다면평가시스템_기획개요서.docx     의사결정용 개요서
│   ├── 02_다면평가시스템_구현설계서.docx     내부 개발용 구현 설계서(Word)
│   └── 구현설계서_BARS포함.md                구현 설계서 + 평가 차원 전문(Markdown)
├── design/
│   └── design-system.tokens.json            디자인 토큰(Pretendard·브랜드 #8dc21f·명도분리 팔레트)
└── eval-form/             역량평가서(BARS) 3종 포맷
    ├── 재활치료센터_역량평가서_BARS.html      운영·열람용(가중치 자동계산)
    ├── 재활치료센터_역량평가서_BARS.xlsx      운영용(점수 입력→자동 집계)
    └── 재활치료센터_역량평가서_BARS.docx      결재·인쇄용
```

## 핵심 설계 요약

- **직급(3단계)**: 치료사 → 선임치료사 → 책임치료사
- **직책(3단계)**: 팀매니저 → 파트매니저 → 시니어매니저
- **평가 차원(16개)**: 공통역량 4 · 직무역량 7 · 리더십역량 5 (BARS 5점 척도)
- **다면평가**: 자기(격차용·0%) + 동료 + 팀원(상향) + 상급자, 역량군별 평가자 차등 가중
- **점수 산정**: 평가자 → 역량군 → 종합의 2단계 가중, S~D 등급 환산
- **구현 전제**: GCP 풀스택 · Antigravity

## 비고

문서 내 평가 등급 명칭은 거부감을 줄이기 위해 “개선필요·미흡”을 “향상 기대·성장 필요”로 순화했습니다. 특정 의료기관을 식별할 수 있는 정보는 포함하지 않았습니다.
